# 🎬 primeFlix

Catálogo de filmes desenvolvido com **React** e consumo da **API do TMDB** — explore, busque e salve seus filmes favoritos.

🔗 **[Ver demo online](https://prime-flix-sigma-bice.vercel.app)**

---

## ✨ Funcionalidades

- Listagem de filmes em cartaz e populares
- Busca por título
- Página de detalhes do filme
- Salvar filmes favoritos
- Interface responsiva

---

## 🛠️ Tecnologias

| Tecnologia | Uso |
|-----------|-----|
| React | Interface de usuário |
| JavaScript | Lógica da aplicação |
| React Router DOM | Navegação entre páginas |
| TMDB API | Dados dos filmes |
| CSS | Estilização |

---

## 🚀 Rodando localmente

```bash
# Clone o repositório
git clone https://github.com/Abderson/primeFlix

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm start
```

Acesse em: http://localhost:3000
